웹가맹점 온라인 가입

  • 2020년부터 웹가맹사가 변경되었습니다. [나이스페이먼츠 → 신한카드(신한PG)] 기존 시설회원은 원활한 결제/정산을 위해 아래 안내에 따라 웹가맹점 변경절차를 진행하여 주시기 바랍니다.

웹가맹점 등록

  • ① 홈페이지 로그인 후 MY페이지 ☞ 웹가맹점 신규/변경
  • ② 이용관련 동의 ☞ 관리자 인증 ☞ 계좌정보 입력

    * 계좌정보 입력시, 사업자등록증상 대표자 또는 상호명의 계좌입력 필요

    * 실제 계좌(통장)정보와 동일하게 입력 필수(예금주, 계좌번호, 은행명)

    * 단, 예금주명은 최대 8자리까지 입력

  • ③ 웹가맹점 등록확인 : ‘MY페이지-시설신청현황’ 가맹점등록여부 확인

    * 처리기한 : 약 3일 이내

웹가맹점 정보변경

  • 시설대표자/주소 변경 : 해당 시군구 담당자에게 변경사항 알림(사업자등록증 송부)
  • 시설정보 변경 후 웹가맹점 정보변경 신청 (My 페이지 -> 웹가맹점 신청/변경)
